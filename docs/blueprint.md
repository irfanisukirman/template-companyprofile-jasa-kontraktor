# **App Name**: Karya Abadi Kontraktor

## Core Features:

- Static Page Content: Display core company information across sections like Hero, About Us, Services, Advantages, Testimonials, and Footer.
- Dynamic Project Portfolio: Showcase construction projects with an interactive image gallery and hover effects, with an option to display before/after views.
- Lead Capture Form: A dedicated contact form and integrated WhatsApp button for potential clients to easily submit inquiries and connect with the contractor.
- Responsive Design: Ensures the website provides an optimal viewing and interaction experience across all devices, from mobile phones to desktop computers (mobile-first approach).
- Enhanced Site Navigation: Implement a sticky navigation bar for consistent access to main sections and smooth scrolling for a seamless user experience.

## Style Guidelines:

- Primary color: A bold orange (#D97706) for brand recognition and energy, derived from the user's preference for 'Orange'.
- Background color: A very light, warm off-white (#F9F1ED) to maintain a clean aesthetic and provide an inviting canvas for content, subtly aligning with the primary hue.
- Accent color: A soft red-orange (#E07070) to highlight interactive elements and calls to action, providing subtle contrast while remaining harmonious.
- Supporting colors: Dark navy (#0B1A2B) for prominent text and contrasting sections, paired with clean white and light gray backgrounds for overall clarity, as per user's guidance.
- Heading font: 'Poppins' (sans-serif) for its modern, geometric appearance, ensuring strong visual impact for headlines.
- Body font: 'Inter' (sans-serif) for exceptional legibility and a contemporary, neutral feel, suitable for extensive body text.
- Use clean, minimalistic icons that visually represent services and features, maintaining a professional and modern look consistent with the site's overall design.
- Utilize generous whitespace and ample spacing between elements to create a clean, uncluttered interface and enhance readability, ensuring an open and breathable design.
- Implement smooth hover effects on interactive elements and seamless scroll animations to enhance user engagement and provide a fluid browsing experience.
